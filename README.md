1.) Keshav Shop – Merchandise Store SQL Project

## 📌 Overview

**Keshav Shop** is a merchandise store designed for developers and coders, offering products such as hoodies, mugs, stickers, notebooks, and accessories.

This project simulates a real-world **e-commerce database system**, focusing on how businesses manage customers, products, orders, and payments using SQL.

---

## 🎯 Objective

The goal of this project is to demonstrate practical SQL skills by building a structured database and performing business-oriented analysis on realistic data.

---

## 🗂️ Key Features

* 🧱 **Database Design**
  Well-structured relational database with normalized tables

* 🔗 **Table Relationships**
  Proper use of primary and foreign keys to maintain data integrity

* 📊 **Realistic Demo Data**
  Simulated real-world e-commerce data for meaningful analysis

* 📈 **Business Analytics Queries**
  SQL queries designed to answer real business questions

---

## 🧩 Database Structure

The system includes the following core entities:

* **Customers** → Stores user details
* **Products** → Stores product catalog
* **Orders** → Tracks customer orders
* **Order Items** → Stores items within each order
* **Payments** → Tracks payment details

---

## 🔗 Relationships

* One customer → Multiple orders
* One order → Multiple products
* Products linked via order items
* Orders linked with payments

---

## 📊 Business Insights Covered

This project enables analysis such as:

* 💰 Total revenue generated
* 🛒 Best-selling products
* 👥 Top customers by spending
* 📦 Order distribution and trends
* ❌ Cancelled orders analysis

---

## ⚙️ How to Run

1. Execute SQL files in order:

   * Run `keshavshoptabledesign.sql`
   * Run `keshav-shop-populate-tables.sql`
   * Run `keshav-shop-running-queries.sql`

---

## 🛠️ Technologies Used

* MySQL
* SQL (Joins, Aggregations, Subqueries)

---

## 🚀 Learning Outcomes

* Designing relational databases
* Implementing foreign key relationships
* Writing analytical SQL queries
* Solving real-world business problems using data

---

## 📌 Future Enhancements

* Add inventory management
* Integrate user authentication
* Build dashboard using Power BI/Tableau
* Add stored procedures & triggers

2.) # Employee Management SQL Project

## 📌 Overview

This project simulates a real-world employee database system used for managing employee data, salaries, departments, and performance.

## 🗂️ Database Structure

* Employees
* Departments
* Salaries (historical data)
* Performance (latest ratings)

## 🔥 Key Features

* Complex JOIN queries
* Subqueries and aggregations
* Window functions (ROW_NUMBER, LAG)
* Stored procedures
* Views

## 📊 Sample Use Cases

* Find top 3 employees per department
* Analyze salary growth
* Identify top performers
* Department-wise salary analysis

## ⚙️ How to Run

1. Run `employee-db-design.sql`
2. Run `employee-db-insert-data.sql`
3. Run `employee-db-run-query.sql`

## 🚀 Technologies Used

* MySQL
* SQL (Advanced Queries)
